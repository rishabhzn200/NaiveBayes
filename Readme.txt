The test dataset used here is the Pima Indians Diabetes problem which can be found at https://archive.ics.uci.edu/ml/machine-learning-databases/pima-indians-diabetes/pima-indians-diabetes.data.

It contains 768 observations of medical details for Pima Indians patients. Each record is classified as 1 or 0 depending on whether the patient has suffered from diabetes within 5 years of when the measurements were taken.

Naive Bayes algorithm is used to create the model and predict the accuracy of test dataset which comes to around 77.49%
